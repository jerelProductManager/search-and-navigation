📖 B&H Data Management — User Guide
Overview ▶

The Data Management page is the central place for maintaining the navigation structure used across B&H's category-browsing tools. From here you can manage two types of data:

  • Navigation Categories — the top-level departments (e.g. Photography, Computers) that appear in the green nav bar.
  • Child Category Labels — sub-section labels within each department, used when assigning Label-type featured slots in the merch tool.

All changes are stored in your browser and instantly reflected in the Nav Preview and Category Detail pages — no page reload required.

Navigation Categories ▶

This table on the left lists every top-level department. Each entry has a Category Name and a CatID — the numeric identifier used by the B&H catalog system.

Edit a category

1Click Edit on the row you want to change. The row turns yellow and both fields become editable.
2Update the Category Name and/or CatID as needed.
3Click Save to apply, or Cancel to discard. You can also press Esc to cancel.

Add a category

1Click + Add Category at the bottom of the table.
2Enter the Category Name (required) and CatID (optional but recommended).
3Click Add or press Enter to confirm.

Remove a category

1Click Remove on the row you want to delete.
2Confirm the prompt. The category is removed immediately from the list.
⚠ CatIDs must be unique. If you try to add a category with a CatID that already exists, the form will show an error.
Child Category Labels ▶

The Child Category Labels section below the two-column area manages the sub-section labels available within each department. Labels appear as selectable options when assigning Label-type featured slots in the Merch Tool.

Select a category

Use the Category dropdown at the top of the section to switch between departments. The table updates immediately to show that department's labels.

Edit a label

1Click Edit on the label row. The row turns yellow.
2Update the Label Name and/or optional ID# field.
3Click Save to apply or Cancel to discard.

Add a label

1Click + Add Label at the bottom of the labels table.
2Enter the Label Name (required) and an optional ID# for system reference.
3Click Add or press Enter to confirm.

Remove a label

1Click Remove on the label row.
2Confirm the prompt. The label is deleted from that category.
💡 Labels belong to a specific category. Switching categories in the dropdown shows only that category's labels — other categories are unaffected.
Saving Changes ▶

Changes you make to categories or labels are held in memory until you explicitly save them. An orange "Unsaved changes" badge appears in the table header whenever there are pending edits.

To save all changes at once: click the Save All Changes button in the sticky bar at the bottom of the page. A green confirmation message will appear briefly.

⚠ Browser storage: All data is saved to your browser's localStorage. Clearing your browser data or switching browsers will reset all changes back to the default seed data. This tool does not connect to a backend server.

Once saved, your changes are automatically reflected in:

  • Nav Preview — the live mega-menu navigation display
  • Category Detail — the per-category featured slots editor
Exporting to Excel ▶

Click the ⬇ Export Excel button in the top-right of the page to download the current data as a .xlsx file named BH_Navigation_Data.xlsx.

The exported workbook contains two sheets:

  • Navigation Categories — all departments with their Category Name and CatID.
  • Child Category Labels — all labels across every department, with columns for Category, CatID, Label Name, and ID#.
💡 The export reflects the current in-memory state, including any unsaved edits. You do not need to click Save All Changes before exporting.
Navigating to Other Pages ▶

The top-right of the page title bar provides quick links to related tools:

  • ← Nav Preview — opens bh-mega-menu.html, the live navigation bar preview that reads from the same localStorage data.
  • Category Detail — opens category-detail.html for the last category you visited, where you can manage featured slots, banners, and other per-category content.

The top sub-navigation bar also provides access to Item Navigation, Saved Reports, Data Management (this page), and Admin.

Tips & Keyboard Shortcuts ▶
  • Press Esc at any time to cancel an active inline edit in either table.
  • Press Enter in the CatID or ID# field of an Add form to confirm adding the new row without clicking the button.
  • Only one row can be in edit mode at a time per table. Starting a new edit automatically cancels the previous one.
  • The + Add Category and + Add Label forms auto-focus their first field when opened.
  • If you accidentally remove a category or label, you can refresh the page before saving to restore the last saved state.
🔄 Resetting to defaults: To restore all original seed data, open your browser's DevTools Console and run: localStorage.removeItem('bh_dm_state'), then refresh the page.
